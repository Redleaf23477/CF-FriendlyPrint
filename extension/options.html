Extension Options

Choose app mode

Normal mode provides an UI in extionsion popup so that user can choose which problems to print when viewing tutorial page.

Works well with most tutorials of codeforces usual rounds.

Dummy mode simply deletes web elements like header, menu bars, side bars, footer, etc.

Users may need to delete other unwanted elements by themselves using something like chrome devtools(press F12 to see it).

For more info, please refer to the readme file.
