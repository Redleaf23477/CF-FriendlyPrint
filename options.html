Extension Options

Choose app mode

Normal mode provides an UI in extionsion popup so that user can choose which problems to print when viewing tutorial page. It works well with most tutorials of codeforces usual rounds.

If users want to print codeforces blog pages other than usual round tutorials, please use dummy mode instead. Normal mode do not support it currently.

Dummy mode simply deletes web elements like header, menu bars, side bars, footer, etc. Users may need to delete other unwanted elements by themselves using something like chrome devtools(press F12 in to see it).
